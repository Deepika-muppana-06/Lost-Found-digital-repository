# CampusFind AI - Lost & Found

CampusFind AI is a campus-based Lost & Found web application built using HTML, CSS, JavaScript, and Firebase. Users can report lost or found items, upload images, and view available items in real-time.

## Features
- User authentication (Sign Up / Login) via Firebase
- Report lost or found items with images
- Store and retrieve item data using Firebase Firestore
- Secure and responsive design
